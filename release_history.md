## v0.0.0.6 - 2026-05-29

### 重大更新

- **VCAN-Views 上线**：USB-CAN 专用免费版本首次发布。与 NexuTrace 同源同步迭代，保留全部数据可视化能力（Hex / Plot / 仪表控件 / 数据面板 / 3D 视图）；专为 USB-CAN 设备调试场景优化，使用门槛更低
- **x86 (32 位) 全面回归**：x64 与 x86 同步发版，单次发版统一交付 4 个安装包
  - NexuTrace x64 / x86
  - VCAN-Views x64 / x86

### 视觉与体验

- NexuTrace 字标横幅重新渲染，连接对话框 Banner 清晰度与抗锯齿质量显著提升
- 标题栏图标在所有产品中正确加载，与任务管理器 / Alt+Tab 切换器图标保持一致
- 资源管理器右键属性 → 详细信息按产品分别显示对应品牌信息
- About 对话框版本号文案自动随产品切换显示
- 双产品独立窗口标识，同机器双开互不冲突

### VCAN-Views 专属优化

- 连接对话框模式选择直接锁定为 USB-CAN（VCAN / Candle 子模式正常切换）
- 工具栏布局精修，无冗余空槽
- 启动流程精简，启动响应更快

### Bug 修复

- 修复部分场景下标题栏图标显示异常（回退到默认字符占位）
- 修复非主产品 Banner 资源加载缺失，连接对话框 Banner 缺图
- 修复 About 对话框版本号文案品牌识别错误
